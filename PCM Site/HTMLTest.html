Web Scraping.

Web Scraping é um processo de extracção de informação de websites que utilizem o protocolo http. O processo de extracção de data pode ser efectuado manualmente ou através de bots e web crawlers.

Tal como um Browser, os web crawlers efectuam a descarga da pagina para que depois possa ser processada num sistema autónomo. (Wikipédia, n.d.)
